# Smart File & Data Management System
A Python-based project to efficiently manage records using a CSV file. Add, view, search, update, and delete records with proper logging and file handling.

---

## Features
- ✅ Add new records (ID, Name, Age, Course)
- ✅ View all records
- ✅ Search records by ID
- ✅ Update existing records
- ✅ Delete records
- ✅ Logging of all function calls
- ✅ Persistent storage in CSV

---

## Project Structure
Smart-File-Data-Management-System/

│

├─ main.py           # Main program and menu

├─ utils.py          # Logger and helper functions

├─ file_handler.py   # CSV file read/write functions

├─ record_manager.py # Record class

├─ records.csv       # CSV file storing records

└─ README.md         # Documentation
